사용자가 설명한 새 기능/대규모 작업에 대해 다음 순서로 진행한다:

## STEP 0 — 선행 조건 체크

`.claude/rules/workflow.md`의 **세션 시작 가드**를 실행한다. 하나라도 실패하면 **즉시 중단**한다.

선행 조건을 모두 통과하면 아래 단계를 진행한다.

---

1. `docs/PRD.md` 존재 여부를 확인한다
   - 있으면: 읽고 현재 제품 명세를 파악한다
   - 없으면: `prd-generator` 에이전트를 호출하여 PRD를 새로 생성한다
2. `prd-generator` 에이전트를 호출하여 PRD에 새 기능 명세를 추가한다 (1단계에서 새로 생성한 경우 건너뛴다)
3. `development-planner` 에이전트를 호출하여 ROADMAP에 Phase/Task를 추가한다
4. 결과 요약을 출력한 후 종료한다 (코드 구현은 하지 않는다)

## 금지 사항

이 커맨드 실행 중 다음 작업을 수행하지 않는다:

- 코드 파일 생성/수정
- 패키지 설치 (`npm install` 등)
- 빌드/서버 실행
- Task 자동 구현
